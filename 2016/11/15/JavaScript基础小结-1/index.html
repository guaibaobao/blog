一、变量：

1.1、什么是变量？

从编程角度讲，变量是用于存储某种、某些数值的存储器。可以讲变量看作一个盒子，用来存放物品，物品可以是衣服、玩具、水果。。。等。
（也可认为是我们数学上曾学习过得代数，例：a=1,b=2, a+b=c a,b,c均为代数，在函数中就是变量）

1.2、变量命名：

变量命名遵循一定规则
1、必须以字母、下划线或美元符号开头，后面可以跟字母、下划线、美元符号和数字。如下:

正确:
mysum
_mychar
$numa1
错误:
6num //开头不能用数字
%sum //开头不能用除(_ $)外特殊符号,如(% + /等)
sum+num //开头中间不能使用除(_ $)外特殊符号，如(% + /等)

2.变量名区分大小写，如:A与a是两个不同变量。
3.不允许使用JavaScript关键字和保留字做变量名。

关键字
break else new var
case finally return void
catch for switch while
default if throw
delete in try
do instanceof typeof

1.3、变量声明：

  • 声明变量方法： var 变量名;

var就相当于找盒子的动作，在JavaScript中是关键字（即保留字），这个关键字的作用是声明变量，并为”变量”准备位置(即内存）。
当然，我们可以一次找一个盒子，也可以一次找多个盒子，所以Var还可以一次声明多个变量，变量之间用”,”逗号隔开。var num1,num2 ;
注意：变量也可以不声明，直接使用，但为了规范，要先声明，后调用。

1.4、 变量赋值：

  • 使用“=”号给变量存储内容（赋值）

例：var mynum = 5; //声明变量mynum并赋值。
另一种写法：var mynum; //声明变量mynum； mynum = 5;//给变量mynum赋值。
注:这里 “=”号的作用是给变量赋值，不是等于号。

1.5 、 表达式

Javascript 语句：num = num + 1 ;

num = num + 1
变量 赋值 表达式

串表达式： (值为字符串)
"I" + "love" + "you" "super" + "mychar"
注意：串表达式中mychar是变量。

数值表达式：（值为数值）
num + 5*3 2+2.5
注意：数值表达式中num是变量。

布尔表达式： (值为true或flase)
2 > 3 num == 5 num < 60
注意：布尔表达式中num是变量。

1.5、 +号操作符：

操作符是用于在javascript中指定一定动作的符号。

  • 操作符：
    sum = numa + numb; //其中“=”和“+”都是操作符。
    JavaScript中还有很多这样的操作符，例如，算术操作符(+、-、*、/等)，比较操作符(<、>、>=、<=等)，逻辑操作符(&&、||、！)。
    注意: “=” 操作符是赋值，不是等于

  • “+”操作符
    算术操作符主要用来完成类似加减乘除的工作，在javascript中，“+”不只代表假发，还可以连接两个字符串，例如：
    Mystring = ＂java＂ + ＂script＂;//mystring 的值“javascript”这个字符串

1.7、&&、||、！

  • 1、&&逻辑与操作符
    b>a && b<c //“&&”是并且的意思, 读法”b大于a”并且” b小于c “
    逻辑与操作符值表：
A B A&&B
真（true） 真（true） 真（true）
真（true） 假（false） 假（false）
假（false） 真（true） 假（false）
假（false） 假（false） 假（false）

注意: 如果A为假，A && B为假，不会在执行B; 反之，如果A为真，要由 B 的值来决定 A && B 的值。

  • 2、|| 逻辑或操作符，相当于“或者”，当两个条件中有任一个条件满足，逻辑或运算结果就为真。

逻辑或操作符值表：

A B A或B
真（true） 真（true） 真（true）
真（true） 假（false） 真（true）
假（false） 真（true） 真（true）
假（false） 假（false） 假（false）
  • 3、“！”逻辑非操作符

逻辑非操作符值表：

A ！A
真（true） 假（false）
假（false） 真（true）

若同级的运算是按从左到右次序进行，多层括号有里向外。

1.9、 数组（一起组团）

★ 变量用来存储数据，一个变量只能存储一个内容，一个数组变量可以存放多个数组，好比一个团，团里有很多人。
数组是值的集合，每个值都有一个索引号（下标），从0开始，每个所以呢都有一个相应的值，根据需要添加更多的值。数组中每个索引，都有一个相应的值。
★ 创建数组（组团，并给团取个名）
使用数组前首先要创建，需要将数组本身赋至一个变量。
语法：var myarray = new Array();
创建数组时，还可以为数组指定长度，长度可任意指定。
var myarray = new Array(8); //创建数组，存储8个数据。
注意：
1.创建的新数组是空数组，没有值，如输出，则显示undefined。
2.虽然创建数组时，指定了长度，但实际上数组都是变长的，也就是说即使指定了长度为8，仍然可以将元素存储在规定长度以外。
★ 数组赋值（谁是团里的成员）
第一步：创建数组var myarr = new Array();
第二步：给数组赋值：myarr[1] = “zhang”;
myarr[2] = “li”;
注意：数组每个值有一个索引号（下标），从0开始。
还可以用简单的方法创建数组和赋值：
①：var myarr = new Array(zhang, li); //创建数组同时赋值
②：var myarr = [zhang, li]; //直接输入一个数组（称“字面量数组”）
数组存储的数据可以是任何类型（数字、字符、布尔值等）。
★ 添加元素：向数组增加一个新元素（团里添加新成员）
添加一个新的索引（下标），任何时刻可以不断上午向数组增加新元素。
myarr[3] = “wang”; //使用新索引（下标），为数组增加一个新元素。
★ 使用数组元素（呼叫团里成员）
得到一个数组元素的值，秩序引用数组变量并提供一个索引，例：
第一个人的姓氏：myarr[0];
第二个人的姓氏：myarr[1];
★ 数组属性length（了解团内成员数量）
Length属性表示数组的长度，技术组重元素的个数。
语法：myarr.length; //获取数组myarr的长度。
注意：因为数组的索引（下标）是从0开始的，所以数组的上下限分别是、；0和length-1。如数组的长度是5，数组的上下限分别是0和4。同时javascript 数组的length 属性是可变的。

★二维数组：
一维数组可以看成一组盒子，每个盒子只能存放一个内容。
一维数组表示：myarray[ ];
二维数组，可以看做一组盒子，但每个盒子里还可以放多个盒子。
二维数组表示：myarray[ ] [ ];
注意: 二维数组的两个维度的索引值也是从0开始，两个维度的最后一个索引值为长度-1。
1、 二维数组的定义方法：

将上面的二位数组，用表格的方式表示：

0 1 2
0 myarr[0][0]值：0 myarr[0][1]值：1 myarr[0][2]值：2
1 myarr[1][0]值：1 myarr[1][1]值：1 myarr[1][2]值：3

2、 二维数组的定义方法二：
var Myarr = [[1,2,3],[1,2,3]];
3、 赋值：
myarr[0][1] = 5; //将5的值传入到数组中，覆盖原有值。
说明：myarr[0][1],0表示表的行，1表示表的列。
